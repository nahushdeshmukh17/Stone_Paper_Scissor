# Stone Paper Scissor Game

A simple interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## Features

- Interactive UI with clickable choice buttons
- Real-time score tracking for user and computer
- Visual feedback with background color changes
- Responsive design with hover effects

## How to Play

1. Open `index.html` in your browser
2. Click on Stone, Paper, or Scissor to make your choice
3. The computer will automatically make its choice
4. Winner is determined by classic rules:
   - Stone beats Scissor
   - Paper beats Stone  
   - Scissor beats Paper
5. Scores are tracked and displayed in real-time

## Files

- `index.html` - Main game interface
- `style.css` - Game styling and layout
- `app.js` - Game logic and functionality
- `images/` - Game choice icons (rock.png, paper.png, scissors.png)

## Game Rules

- Stone crushes Scissor
- Paper covers Stone
- Scissor cuts Paper
- Same choices result in a draw

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
